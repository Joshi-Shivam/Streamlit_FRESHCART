# FreshCart 🛒

FreshCart is an interactive grocery shopping application built using Streamlit. The application allows users to browse products from multiple grocery categories, add items to their cart, review purchases in real time, and generate a downloadable PDF bill after checkout.

## Features

### 🥬 Multiple Product Categories

* Fruits
* Vegetables
* Dairy Products
* General Household Items

Users can browse one category at a time through an intuitive and visually appealing interface.

### 🖼️ Product Images

Each product category contains relevant product images to provide a realistic online shopping experience.

### 🎚️ Quantity Selection

Users can select the quantity of each item using interactive sliders with configurable limits.

### 🛒 Live Cart Updates

The cart updates dynamically as users select quantities. Selected products and quantities are displayed instantly in the sidebar.

### 📊 Real-Time Cart Summary

The application continuously tracks selected products and their quantities, providing users with a live overview of their shopping cart.

### 💰 Automatic Price Calculation

FreshCart automatically calculates item costs based on selected quantities and predefined product prices.

### 🧠 Session State Management

The application uses Streamlit Session State to maintain user selections, cart contents, category navigation, and checkout information throughout the session.

### 🎉 Interactive User Experience

* Balloons animation on successful checkout
* Snow effects for enhanced user engagement
* Responsive and interactive Streamlit interface

### 📄 PDF Bill Generation

After checkout, FreshCart generates a professional PDF invoice containing:

* Customer Name
* Bill Number
* Date and Time
* Purchased Items
* Quantities
* Item Prices
* Total Bill Amount

### ⬇️ PDF Download Support

Users can instantly download their generated bill in PDF format directly from the application.

### 🎨 Modern User Interface

* Custom Streamlit theme
* Organized category layout
* Sidebar cart management
* Clean and user-friendly design
